# Web Infrastructure Design

This project contains the design and explanation of four web infrastructure architectures.

## Diagrams

- **Task 0** - [Simple Web Stack](./0-simple_web_stack.md): Single server architecture with Nginx, App Server, Code, and MySQL
- **Task 1** - [Distributed Web Infrastructure](./1-distributed_web_infrastructure.md): Distributed architecture with HAproxy, 2 servers, and Primary-Replica MySQL
- **Task 2** - [Secured and Monitored Web Infrastructure](./2-secured_and_monitored_web_infrastructure.md): Secured architecture with firewalls, SSL, and monitoring clients
- **Task 3** - [Scale Up](./3-scale_up.md): Scaled architecture with separated tiers and clustered load balancers
