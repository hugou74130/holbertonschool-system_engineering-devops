# Web Infrastructure Design

This project contains the design and explanation of four web infrastructure architectures.

- [Task 0 - Simple Web Stack](task0_simple_web_stack.png)
- [Task 1 - Distributed Web Infrastructure](task1_distributed.png)
- [Task 2 - Secured and Monitored Web Infrastructure](task2_secured_monitored.png)
- [Task 3 - Scale Up](task3_scale_up.png)

## Files

| File | Description |
|------|-------------|
| `0-simple_web_stack` | Single server architecture with Nginx, App Server, Code, and MySQL |
| `1-distributed_web_infrastructure` | Distributed architecture with HAproxy, 2 servers, and Primary-Replica MySQL |
| `2-secured_and_monitored_web_infrastructure` | Secured architecture with firewalls, SSL, and monitoring clients |
| `3-scale_up` | Scaled architecture with separated tiers and clustered load balancers |
